# 需求文档 - 规则配置

## 现状

业务规则（计费、折扣、生效周期等）目前分散在 Excel 或口头约定中，运营需在多个系统间核对配置，缺乏统一入口。

## 痛点

- 配置查找慢，无法按客户快速筛选
- 草稿与已发布状态不清晰，误操作风险高
- 新建/编辑流程不统一，缺少标准表单结构

## 诉求与目标

提供统一的「规则配置」管理界面：支持列表检索、详情查看、表单新建/编辑，状态流转清晰（草稿 → 发布 → 作废）。

## 范围边界

### 做

- 列表页：筛选、表格、行操作（查看/编辑/发布/删除/作废/复制）
- 表单页：分区字段、暂存/发布
- 详情弹窗：只读展示

### 不做

- 真实后端 API 对接
- 复杂审批流
- 权限体系

## 验收标准

- [ ] 列表右侧规则区：检索条件下有「新增」按钮；列含规则名称、创建人/时间、更新人/时间
- [ ] 列表可按客户、生效状态、创建人、创建/更新时间、更新人筛选
- [ ] 草稿行可编辑/发布/删除，已发布行可查看/作废
- [ ] 新增/编辑跳转表单页，查看打开详情弹窗
- [ ] 删除/复制操作有二次确认
- [ ] 选择价卡后展示三张费用卡片（快递费 / 库内费用 / 其他费用），默认带出价卡费用项，支持增删行
- [ ] 同客户 published 规则生效期不可重叠；发布前展示影响预览弹窗
- [ ] 列表展示生效状态；发布时冻结规则快照

## 变更记录

### 2026-07-07 · 规则列表工具栏与列字段

新增按钮移入规则列表区；规则名称列；创建人/创建时间/更新人/更新时间为独立列。

### 2026-07-07 · 规则列表内嵌检索与列调整

检索条件移入右侧规则列表区；删除价卡来源列，恢复创建/更新信息列。

### 2026-07-07 · 列表左右分栏展示

主表改为左侧客户列表 + 右侧规则列表：选中客户后展示各有效期周期的价卡来源、配置名称与状态，便于按客户浏览「什么时间用什么价卡」。

### 2026-07-07 · 列表主表 UX 优化与详情页表单对齐

列表主表移除冗余「客户」列，合并审计字段为「创建信息」「更新信息」，默认展开树形子行；详情页采用与新增/编辑表单相同的分仓 sidebar + 费用导航 + 只读表格布局，mock 数据与表单示例一致。

### 2026-07-07 · 详情弹窗改为独立详情页

将列表页「查看」操作从详情弹窗改为跳转独立详情页，详情页内容结构调整为与表单页对齐（基础配置、仓库信息、费用配置采用 Tab 结构、其他信息、发布状态），费用配置区采用 Tab 展示方式，保持与表单页的一致性，提升用户查看体验。

### 2026-07-07 · 列表树形结构与仓库列移除

列表主表改为按客户分组的树形结构：一级展示客户及「共 N 个有效期」汇总，二级展示各有效期配置，默认折叠。删除主表仓库列，行操作仅对有效期子行生效。生效期重叠规则调整为同客户维度不可重叠；配置名称不含仓库。详情弹窗基本信息仍保留仓库字段。

### 2026-07-06 · 快速填入交互优化

将费用配置区的快速填入功能从工具栏分段控件改为仓库列表按钮触发弹窗，采用三步向导流程（选择复用类型→选择来源→确认覆盖），简化操作步骤，提升用户体验。应用前显示覆盖警告提示，降低误操作风险。

### 2026-07-05 · 规则生效逻辑优化

采用时间轴版本化：闭区间匹配（结束默认 23:59:59，新 start 默认次日 00:00:00）、published 不可原地修改、发布前重叠校验与影响预览、立即生效自动截断、发布冻结规则快照。

### 2026-07-05 · 移除审批时间轴

新增/编辑表单页不再展示审批时间轴（Extension）区域。

### 2026-07-05 · 引用价卡后三张费用卡片

新增/编辑规则配置时，选择价卡模板后在「引用价卡」与「其他信息」之间展示三张费用卡片（快递费 / 库内费用 / 其他费用），结构与价卡管理一致。默认根据所选价卡带出费用规则行，支持增删行及折扣设置。详情弹窗费用展示与表单卡片结构对齐。

### 2026-07-04 · 删除与复制二次确认

列表页删除、复制操作增加确认弹窗，降低误操作风险。
