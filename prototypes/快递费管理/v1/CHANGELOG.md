## 2026-07-04
- **新增名称属性**
  - 表单页新增「名称」字段（必填文本输入）
  - 列表页表格新增「名称」列
  - 详情弹窗新增「名称」显示
  - 编辑时自动预填充名称数据
- **编辑状态下费用类型不可编辑**
  - 编辑模式下，费用类型字段自动禁用，不允许修改
  - 新增模式下，费用类型字段可正常选择
- **重量段拆分为起止两列**
  - 重量段拆分为「起」和「止」两个列
  - 「起」自动继承上一行的「止」，不允许编辑
  - 只有「止」列可编辑
  - 编辑「止」后自动更新下一行的「起」
  - 删除行后自动重新对齐后续行的起止重量
- **分区报价表格交互逻辑**
  - 添加重量段按钮：点击后新增一行，开始重量自动填充上一行结束重量
  - 操作列删除按钮：点击后删除当前行
  - 导入分区报价按钮：点击后弹出导入弹窗，支持上传 Excel/CSV 文件
  - 单价输入框：支持实时编辑价格数值
- **分区报价表格移至表单页**
  - 分区报价配置从详情弹窗移到表单页（编辑页面）
  - 详情弹窗仅用于查看基本信息（只读）
  - 新增/编辑时在表单页统一维护基础信息 + 分区报价表格
- **编辑表单预填充支持**
  - 扩展 schema 支持 formField.defaultValue 和 page.editData
  - 编辑时表单字段自动预填充已有数据（渠道、承运商、仓库、费用类型）
  - RuleConfigFormPattern 组件支持 editId 获取预填充数据
- **分区报价表格编辑支持**
  - 扩展 schema 支持 zone-price-table layout 和 formSection.zonePriceTable
  - 新增 ZonePriceTableEditor 组件，支持表格编辑分区报价
  - 支持添加重量段、编辑单价、删除行操作
- **初始化基础价管理原型**
  - 列表页：支持按渠道、承运商、仓库、费用类型、创建人筛选
  - 列表页：行操作（查看、编辑、复制、删除）
  - 表单页：新增/编辑基础信息 + 分区报价配置
  - 详情弹窗：展示基本信息（只读查看）
