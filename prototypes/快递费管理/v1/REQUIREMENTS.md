# 需求文档 - 快递费管理

## 现状

产品管理人员通过手动整理方式管理海外仓快递费公报价，基础价记录分散在表格中。

## 痛点

- **无法归档溯源**：历史报价无系统记录，难以追溯
- **工作量大**：每次更新需手动整理，效率低且易出错

## 诉求与目标

- 建立系统化的快递费公报价管理能力
- 支持基础价的新增、批量导入、编辑、删除、复制
- 实现归档溯源，提升配置效率

## 范围边界

### 做

- 基础价配置：列表、新增、编辑、删除、复制、批量导入、导出
- 基础信息：渠道、承运商、仓库、费用类型
- 分区报价配置：分区（2-8区）、重量段、单价
- 分区报价表格编辑：添加重量段、编辑单价、删除行
- 附加费/其他费用配置：附加费管理区域，包含属性费用项、计费单位、单价、分区（多选）、起始重量、结束重量、备注
- 附加费管理表格编辑：添加行、删除行、导入配置
- 费用类型联动：选择不同费用类型展示不同的配置区域

### 不做

- 状态管理（草稿/生效/失效）

## 验收标准

- [ ] 列表可按渠道、承运商、仓库、费用类型、创建人筛选
- [ ] 列表页表格显示名称列（第一列）
- [ ] 行操作支持查看、编辑、复制、删除
- [ ] 新增跳转表单页，名称字段为空，其他基础信息字段为空，分区报价表格为空
- [ ] 编辑跳转表单页，名称字段自动预填充，其他基础信息字段自动预填充，分区报价表格显示已有数据
- [ ] 编辑模式下，费用类型字段不允许编辑（禁用状态）
- [ ] 新增模式下，费用类型字段可正常选择
- [ ] 表单页包含名称 + 基础信息 + 分区报价配置三部分
- [ ] 详情弹窗显示名称（基本信息第一项）
- [ ] 分区报价表格重量段拆分为「起」「止」两列
- [ ] 「起」自动继承上一行「止」，不允许编辑
- [ ] 只有「止」列可编辑，编辑后自动更新下一行「起」
- [ ] 分区报价表格支持添加重量段（点击按钮新增一行，开始重量自动填充上一条结束重量）
- [ ] 分区报价表格支持删除重量段行（点击操作列删除按钮删除当前行，删除后自动对齐后续行）
- [ ] 分区报价表格支持导入功能（点击导入按钮弹出导入弹窗）
- [ ] 分区报价表格单价支持实时编辑
- [ ] 详情弹窗仅展示基本信息（只读查看）
- [ ] 选择「附加费」或「其他费用」时，表单下方展示附加费管理区域
- [ ] 选择「基础价」时，表单下方展示分区报价配置
- [ ] 附加费管理区域包含：属性费用项、计费单位、单价、分区（多选）、起始重量、结束重量、备注
- [ ] 附加费管理区域支持添加行、删除行、导入配置
- [ ] 列表包含附加费和其他费用类型的 mock 数据
